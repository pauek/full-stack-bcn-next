Lists use 3 tags:

|        |                                                                                                               |
| ------ | ------------------------------------------------------------------------------------------------------------- |
| `<ol>` | **o**rdered **l**ist ([mdn&UpperRightArrow;](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ol))   |
| `<ul>` | **u**nordered **l**ist ([mdn&UpperRightArrow;](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ul)) |
| `<li>` | **l**ist **i**tem ([mdn&UpperRightArrow;](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/li))      |

### Ordered Lists

```html
<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
  <li>Fourth</li>
</ol>
```

### Unordered lists

```html
<ul>
  <li>Apples</li>
  <li>Oranges</li>
  <li>Kiwis</li>
  <li>Pears</li>
</ul>
```

Both types of lists can have these attributes:

|            |                                                    |
| ---------- | -------------------------------------------------- |
| `type`     | The type of the list: `1`, `a`, `A`, `i`, `I`      |
| `reversed` | Shows the list reversed                            |
| `start`    | Sets the index of the first item (for split lists) |
